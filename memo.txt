---------------- comparisonVer001 웹으로 이식하기 위한 준비


  1. 비교할 엑셀 파일을 서버에 저장하기
  파일 업로드 버튼 클릭 시  
  static/uploadedExcelFile

  2. 파일 경로 설정 시
    파일 경로 설정 
      file_a = "static/uploadedExcelFile/" + "1stFileName"
      file_b = "static/uploadedExcelFile/" + "2ndFileName"
    이런 식으로 작성할 것이기 때문에
    json 데이터를 html 단에서 받아서 이쪽으로 넘길 것이다.(js)
  
  3. 기준이 되는 컬럼 정하기, 매칭하기
    3.1. 선택을 위해 업로드된 엑셀 파일에서 추출할 것들
      시트가 여러장인 경우 시트 선택
      각 시트마다 첫번째 행/또는 열 선택하여 추출하여 사용자가 선택하게 하기
      선택한 행/열은 이하 "기준 행/열" 로 한다
        main, sub 기준 하나씩 설정할 수 있게 하자.
      각 시트마다 기준 행/열의 모든 컬럼 조회하여 비교하기
    
    3.2. 각 파일마다 나온 결과를 서로 매핑해주기
      ex) a_main_sub : b_sheet001 main
          a_sheet001_main_sub : ....


  4. 데이터 정제하기
    기본
      기준 컬럼의 공백 제거
      모든 줄바꿈 제거
      모든 특수문자 제거
    
    선택
      한글만 남기기
      한자 제거
      영어 제거

  5. 출력 데이터 설정하기
    기본 및 고정
      A에는 있지만 B에는 없는 자료
      B에는 있자만 A에는 없는 자료
      사용자가 기준으로 정한 행/열 컬럼 이름으로

    추가는 사용자가 선택할 수 있게
      추출한 기준 행/열의 컬럼에서만 선택 가능하게 한다.
